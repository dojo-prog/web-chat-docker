# Web Chat Application

A real-time **web chat application** for seamless messaging and communication.
## Features

- **Real-time Messaging:** Messages appear instantly for all participants.  
- **User Authentication:** Secure login and registration system.  
- **Message History:** Access previous conversations with persistence.  

## Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express
- **Database:** MongoDB   
- **Real-time Communication:** Socket.IO / WebSockets  
- **Containerization:** Docker & Docker Compose
